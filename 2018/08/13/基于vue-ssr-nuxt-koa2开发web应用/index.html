Posted on 2018-08-13

前言

如果你的业务既需要SPA的强交互性，又对SEO和首屏加载有刚性需求，作为前端工程师，又想通过BFF架构掌握更多主动权。
但是又不敢轻易尝试，可以参考一下我的经历。

一开始我是想基于vue cli + element-ui快速开发上线，得知需要支持SEO后，打算加一个prerender来做的。
后来同事建议可以尝试BFF架构，在node层做一些BI报表类的功能，加之综合考察后，发现国内已经有一些基于nuxt的线上项目，
于是决定使用nuxt结合koa2来做。
